    class Kernel¶

    SPICE Toolkit kernel. Generalized data class for every type of kernel.

    https://naif.jpl.nasa.gov/pub/naif/toolkit_docs/C/info/intrdctn.html

    enumerator SCLK¶

    enumerator IK¶
