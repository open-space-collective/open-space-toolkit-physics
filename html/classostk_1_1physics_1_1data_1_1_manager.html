Detailed Description

OSTk Data manager base class (thread-safe) 

                        The base manager defines methods for tracking and checking the manifest file.

◆ checkManifestAgeAndUpdate_()

void ostk::physics::data::Manager::checkManifestAgeAndUpdate_ ( ) const
protected

Check the age of the manifest to determine if it should be updated before querying it for other data files. 

                    We fetch the manifest file from the remote if:
                    - It does not exist locally
                    - We are past the next predicted update for the manifest based on its own "manifest"
                    entry.

Determines how often OSTk should do non-critical IO to freshen its data. We limit this by limiting how frequently the manifest file is updated.

Returns
Frequency in hours. Default is 24. Overriden by the OSTK_PHYSICS_DATA_REFRESH_RATE environment variable.
