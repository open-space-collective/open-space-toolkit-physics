◆ computeApArray()

Unique< NRLMSISE00::ap_array > ostk::physics::environment::atmospheric::earth::NRLMSISE00::computeApArray ( const Instant &  anInstant) const
protected

Fill the provided array with AP values needed for the NRLMSISE model. The output array is filled like so:

0 : daily AP [average of current day AP values] 1 : 3 hr AP index for instant 2 : 3 hr AP index for 3 hrs before instant 3 : 3 hr AP index for 6 hrs before instant 4 : 3 hr AP index for 9 hrs before instant 5 : Average of eight 3 hr AP indices from 12 to 33 hrs prior to instant 6 : Average of eight 3 hr AP indices from 36 to 57 hrs prior to instant

Returns
