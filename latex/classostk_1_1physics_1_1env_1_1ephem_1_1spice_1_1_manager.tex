\doxysubsection{Detailed Description}
\mbox{\hyperlink{classostk_1_1physics_1_1env_1_1ephem_1_1_s_p_i_c_e}{S\+P\+I\+CE}} Toolkit kernel manager. 

\begin{DoxyVerb}                        Fetches and manages necessary SPICE kernels.

                        The following environment variables can be defined:

                        - "OSTK_PHYSICS_ENVIRONMENT_EPHEMERIDES_SPICE_MANAGER_LOCAL_REPOSITORY" will override
                        "DefaultLocalRepository"
                        - "OSTK_PHYSICS_ENVIRONMENT_EPHEMERIDES_SPICE_MANAGER_REMOTE_URL" will override
                        "DefaultRemoteUrl" 
\end{DoxyVerb}
