$<$/details$>$\hypertarget{index_autotoc_md11}{}\doxysection{Data Management}\label{index_autotoc_md11}
O\+S\+Tk Physics uses input data from various sources to determine the state of the space environment at any given time. The following input data is used\+:


\begin{DoxyItemize}
\item I\+E\+RS Reference Frame Data
\item C\+S\+SI Solar Flux/\+Space Weather Data
\item Gravitational Survey Data
\item Magnetic Survey Data
\item S\+P\+I\+CE Kernels
\end{DoxyItemize}

None of these files are shipped with the source code of this library. O\+S\+Tk Physics has the capability to fetch the required files at runtime if they are not present or if they are outdated. This is done using file Manager classes (see any file named {\ttfamily Manager.\+hpp}). Data for any use-\/case is queried through the Manager class rather than directly, which allows the Manager to handle file loading and fetching.
